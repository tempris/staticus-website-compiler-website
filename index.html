This is Staticus!

Build Websites The Staticus Way: Controlled, Optimized, Static.

Staticus Website Compiler empowers developers who love writing HTML, CSS, and JavaScript by providing client-side automation and optimization features without the need for complex frameworks. Whether you're building static websites, landing pages, or blogs, Staticus streamlines your workflow and enhances performance.

Why March with Staticus?

Automated Build Process

Leverage an intelligent task management system powered by Gulp to automate tasks like:

  • Minification of HTML, CSS, and JS
  • Image optimization with watermarking capabilities
  • Font icon generation from SVG files
  • Auto-injection of metadata and table of contents
  • Real-time file watching and live reloading

User-Friendly GUI

Manage your project with ease using our intuitive graphical interface, built with CustomTkinter, offering:

  • Directory management with recent project history
  • One-click task execution
  • Visual task categorization and descriptions

Seamless Packaging

Effortlessly package your website for deployment with:

  • Automatic removal of development-specific content
  • Compression into a deploy-ready ZIP file
  • Fine-tuned exclusion of unnecessary assets

Cross-Platform Compatibility

Whether you're on Windows or Linux, Staticus ensures smooth operation and installation through automated dependency handling.

Our Arsenal

(Key Features)

Build Automation

  • Minify and optimize assets for production
  • Customizable configuration with JSON-based site settings
  • Advanced file watching to detect and rebuild changes instantly

Dynamic Content Handling

  • Auto-generate breadcrumbs and structured navigation
  • Automated sitemap generation for SEO improvements
  • Configurable depth for in-page Table of Contents (TOC)

Smart Asset Management

  • Intelligent caching to avoid redundant builds
  • File exclusion rules to keep your output clean
  • Integration with GraphicsMagick for advanced image processing

Easy Deployment

  • Ready-to-ship ZIP packages
  • Automated versioning and asset bundling
  • Lightweight setup with minimal dependencies

The Staticus Strategy

(How It Works)

  1. Set Up Your Project:
    Point Staticus to your project directory and it will automatically configure itself.

  2. Run Tasks Effortlessly:
    Use the GUI to build, watch, and package your project with a click.

  3. Deploy with Confidence:
    Generate an optimized ZIP ready for production with development files stripped out.
